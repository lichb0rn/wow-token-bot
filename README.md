# World of Warcraft discord bot

## Setup

### Getting tokens

First, you need to get all tokens and ids from appropriate developer portals:

- Battle.net:

  - <https://develop.battle.net>
  - Battle.net client id
  - Battle.net secret key

- Discord:

  - <https://discord.com/developers>
  - Discord bot token
  - Discrod app id
  - Discrod guild id
